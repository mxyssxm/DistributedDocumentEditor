# DistributedDocumentEditor

## Description
DistributedDocumentEditor est un projet de système distribué permettant la modification collaborative en temps réel d’un document, avec synchronisation des modifications entre plusieurs nœuds.

## Objectifs du projet
- Comprendre les principes des systèmes distribués
- Gérer les modifications concurrentes d’un document
- Assurer la cohérence des données entre plusieurs clients
- Mettre en place une communication réseau fiable

## Fonctionnalités
- Édition simultanée d’un document par plusieurs utilisateurs  
- Synchronisation des mises à jour  
- Gestion basique des conflits d’édition  
- Architecture distribuée (client–serveur ou pair-à-pair)
